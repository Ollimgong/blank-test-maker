# BTM (Blank Test Markup) 명세

백지테스트 메이커에서 사용하는 파일 포맷과 마크업 문법.

---

## 1. `.btm` 파일 포맷

`.btm` 파일은 JSON 기반의 단원(시험지) 데이터 파일이다.

### 구조

```json
{
  "title": "수동태",
  "rows": [
    {
      "id": "k3x8m2p1",
      "l": { "text": "# SUMMARY", "indent": 0 },
      "r": { "text": "# PRACTICE", "indent": 0 }
    },
    {
      "id": "a9b2c4d7",
      "l": { "text": "@개념 능동태는 주어가 직접 행동함", "indent": 0 },
      "r": { "text": "! 1 나의 친구들은 나를 사랑한다.", "indent": 0 }
    }
  ]
}
```

### 필드 설명

| 필드 | 타입 | 설명 |
|------|------|------|
| `title` | string | 단원 제목 |
| `rows` | array | 행 배열 (30행 고정) |
| `rows[].id` | string | 행 고유 식별자 (내부용, 랜덤 문자열) |
| `rows[].l` | object | 왼쪽 셀 (1열) |
| `rows[].r` | object | 오른쪽 셀 (2열) |
| `cell.text` | string | 셀 내용. BTM 마크업 포함 |
| `cell.indent` | number | 들여쓰기 레벨 (0/1/2, 현재 미사용) |

### 규칙
- 1파일 = 1단원 (시험지 1장)
- 행 수는 항상 30행 (A4 용지 기준)
- 좌/우 셀은 완전히 독립 (각각 별도의 text)
- 파일 인코딩: UTF-8

---

## 2. BTM 마크업 문법

셀의 `text` 필드 안에서 사용하는 인라인 마크업 문법이다.
시험지(WORKSHEET)에서는 일반 텍스트가 숨겨지고, 마크업 요소와 `! `로 시작하는 줄만 표시된다.

### 줄 단위 문법

줄 전체에 적용되며, 반드시 텍스트의 **맨 앞**에 위치해야 한다.

| 문법 | 이름 | 설명 | 시험지 표시 |
|------|------|------|------------|
| `# 텍스트` | 헤더 | 섹션 구분 헤더. 초록색, 굵은 대문자, 넓은 자간 | 항상 표시 |
| `## 텍스트` | 소제목 | 강조 소제목. 굵은 글씨 + 밑줄 | 항상 표시 |
| `!텍스트` 또는 `! 텍스트` | 보이는 줄 | 시험지에서도 숨기지 않고 표시. (띄어쓰기 유무 무관) | **표시** |

**예시:**
```
# SUMMARY
## 주격관계대명사
## 능동태 -> 수동태 전환방법
! 1 나의 친구들은 나를 사랑한다.
```

**주의:** `## `가 `# `보다 먼저 매칭된다 (## 로 시작하면 소제목, # 로 시작하면 헤더).

### 인라인 문법

텍스트 어디에든 삽입 가능하며, 여러 개를 조합할 수 있다.

| 문법 | 이름 | 설명 | 시험지 표시 |
|------|------|------|------------|
| `@태그이름` | 태그 | 등록된 태그와 매칭되면 컬러 뱃지로 렌더링 | 항상 표시 |
| `[텍스트]` | 라벨 | 가벼운 회색 뱃지. 자유 텍스트 | 항상 표시 |
| (일반 텍스트) | 숨김 텍스트 | 답지에만 표시, 시험지에서 투명 처리 | 숨김 |

**예시:**
```
@개념 주격관계대명사는 관계대명사가 이끄는 절에서 주어의 역할을 한다
[영작] My friends love me.
! 1 나의 친구들은 나를 사랑한다.
능동태의 주어, 동사, 목적어를 찾는다.
```

### 이스케이프

문법 기호를 리터럴(그대로)로 표시하고 싶을 때 사용한다.

| 문법 | 결과 |
|------|------|
| `[[텍스트]]` | `[텍스트]` (라벨로 인식되지 않음) |

### 줄 단위 문법의 조합

`! ` (혹은 빈칸 없는 `!`) 줄 안에서 `@태그`와 `[라벨]`, 심지어 `# 헤더`까지 조합하여 사용할 수 있습니다.

```
!@개념 능동태는 주어가 직접 행동함
![영작] She wrote a letter.
!# 핵심 포인트
```

위 예시에서 `@개념` 태그 뱃지와 텍스트 모두 시험지에서 표시된다.

### 태그 (@)

- 태그는 앱 설정에서 미리 등록한 이름만 매칭된다
- 기본 태그: `개념`, `형태`, `예문`, `주의`, `예시`
- 각 태그는 고유 색상을 가진다
- 등록되지 않은 이름 뒤의 `@`는 일반 문자로 취급된다
- 편집 시 `@`를 입력하면 자동완성 드롭다운이 표시된다

### 렌더링 규칙

| 모드 | 일반 텍스트 | `! ` 줄 | 태그/라벨/헤더/소제목 |
|------|-----------|---------|-------------------|
| 답지 (ANSWER) | 표시 | 파란색 표시 | 표시 |
| 시험지 (WORKSHEET) | 투명 (숨김) | 파란색 표시 | 표시 |

## 3. 예시: 완성된 단원

```json
{
  "title": "수동태",
  "rows": [
    { "id": "a1", "l": { "text": "# SUMMARY", "indent": 0 }, "r": { "text": "# PRACTICE", "indent": 0 } },
    { "id": "a2", "l": { "text": "## 능동태 -> 수동태 전환방법", "indent": 0 }, "r": { "text": "## 영작 & 수동태 전환 연습", "indent": 0 } },
    { "id": "a3", "l": { "text": "@예문 She wrote a letter.", "indent": 0 }, "r": { "text": "! 1 나의 친구들은 나를 사랑한다.", "indent": 0 } },
    { "id": "a4", "l": { "text": "1) 능동태의 주어, 동사, 목적어를 찾는다.", "indent": 0 }, "r": { "text": "[영작] My friends love me.", "indent": 0 } },
    { "id": "a5", "l": { "text": "@주의 A letter was written by she. (X)", "indent": 0 }, "r": { "text": "[수동태] I am loved by my friends.", "indent": 0 } }
  ]
}
```

**답지 렌더링:**
```
┌─────────────────────────────┬─────────────────────────────┐
│ SUMMARY                     │ PRACTICE                    │  ← 헤더
├─────────────────────────────┼─────────────────────────────┤
│ 능동태 -> 수동태 전환방법     │ 영작 & 수동태 전환 연습       │  ← 소제목
│ [예문] She wrote a letter.  │ 1 나의 친구들은 나를 사랑한다. │
│ 1) 능동태의 주어...찾는다.   │ [영작] My friends love me.   │
│ [주의] A letter was...      │ [수동태] I am loved by...    │
└─────────────────────────────┴─────────────────────────────┘
```

**시험지 렌더링:**
```
┌─────────────────────────────┬─────────────────────────────┐
│ SUMMARY                     │ PRACTICE                    │  ← 헤더 (표시)
├─────────────────────────────┼─────────────────────────────┤
│ 능동태 -> 수동태 전환방법     │ 영작 & 수동태 전환 연습       │  ← 소제목 (표시)
│ [예문]                      │ 1 나의 친구들은 나를 사랑한다. │  ← ! 줄 표시
│                             │ [영작]                       │  ← 라벨만 표시
│ [주의]                      │ [수동태]                     │  ← 일반 텍스트 숨김
└─────────────────────────────┴─────────────────────────────┘
```
