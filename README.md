# 백지테스트 메이커

영어학원에서 사용하는 **백지테스트(빈칸 시험지)** 를 웹에서 편집하고 인쇄할 수 있는 도구.

기존에 한글(HWP) 프로그램으로 제작하던 백지테스트를, 누구나 브라우저에서 쉽게 만들 수 있도록 React 단일 파일로 구현.

---

## 현재 기능

### 핵심 기능
- **편집기**: 2단(왼쪽/오른쪽) 테이블 형식으로 내용 입력
- **실시간 미리보기**: 편집 화면 옆에 A4 비율 미리보기 동시 표시
- **답지/시험지 자동 전환**: 각 셀의 "답" 버튼으로 마킹하면 시험지 모드에서 자동 숨김
- **인쇄**: `Ctrl+P`로 A4 인쇄 최적화 (`@page { size: A4 }`)
- **데이터 영속성**: `window.storage` API로 세션 간 자동 저장
- **JSON 내보내기/불러오기**: 백업 및 공유용

### 편집 기능
- **태그 시스템**: 컬러칩 팝업으로 선택 — 숫자태그(1~10)와 일반태그(개념/형태/예문 등) 구분 표시, 설정에서 추가/삭제/색상 변경
- **마커(mark)**: 자유 텍스트 접두어 — `[영작]`, `(1)`, `※` 등 직접 입력
- **헤더(H)**: 셀을 섹션 헤더 스타일로 전환 — 어느 행에서든 배치 가능, 왼쪽/오른쪽 독립
- **굵게(B)**: 섹션 제목용 볼드+밑줄 스타일
- **답안(답)**: 시험지 모드에서 숨길 내용 마킹
- **들여쓰기(┗)**: 레벨 0/1/2 — 윗줄의 태그/마커 너비만큼 투명 복제하여 이어쓰기 정렬
- **왼쪽/오른쪽 독립 이동**: ▲▼로 각 칸의 내용을 독립적으로 위/아래 이동
- **⋯ 메뉴**: H/B/답/┗ 토글을 팝오버로 통합하여 편집 행 UI 간소화

### 구조 관리
- **단원(Unit)**: 하나의 시험지 = 하나의 단원
- **그룹**: 단원들을 그룹으로 분류 (예: "문법 기초", "고급")
- **30행 고정**: A4 노트 느낌의 고정 행 수 (추가/삭제 없음)
- **복제**: 기존 단원을 복제해서 새 단원 생성
- **그룹 간 이동**: 단원 메뉴(⋯)에서 그룹 이동

### 인쇄/출력
- **A4 고정 레이아웃**: 740x1046px, 미리보기와 인쇄가 동일한 레이아웃
- **행 높이 고정(27px)**: 내용이 길어도 행이 늘어나지 않음
- **텍스트 클리핑**: 긴 텍스트는 잘림 처리 (ellipsis)
- **슬로건/로고**: 설정에서 등록하면 하단에 표시 (슬로건 왼쪽, 로고 오른쪽)
- **커스텀 폰트**: woff2/woff/ttf/otf 업로드 지원 (기본: Pretendard)

## 데이터 모델

```javascript
// 전체 상태
{
  groups: [{ id, name, collapsed }],
  units: [{
    id, groupId, title,
    rows: [{               // 30행 고정
      id,
      l: { tag, mark, text, vis, bold, hdr, indent },  // 좌우 동일 구조
      r: { tag, mark, text, vis, bold, hdr, indent }
    }]
  }],
  settings: { logo, slogan, customFont, customFontName, tags, numTagColor }
}
```

### 셀 속성 설명
| 속성 | 설명 |
|------|------|
| `tag` | 컬러 태그 — 숫자태그(1~10) 또는 일반태그(개념/형태/예문 등) |
| `mark` | 자유 텍스트 마커 — `[영작]`, `(1)` 등 직접 입력 |
| `text` | 내용 텍스트 |
| `vis` | `true`면 시험지 모드에서도 표시 (false면 숨김) |
| `bold` | `true`면 굵게 + 밑줄 (섹션 제목용) |
| `hdr` | `true`면 헤더 스타일 (초록 배경, 굵은 테두리) |
| `indent` | 들여쓰기 레벨 (0=없음, 1=태그까지, 2=태그+마커까지) |

## 기술 스택

- **React** (JSX 단일 파일)
- **Vite** (로컬 개발서버, 핫 리로드)
- **Pretendard** (웹폰트, CDN)
- **인라인 스타일** (Tailwind 미사용, CSS-in-JS 패턴)
- **window.storage API** (Claude Artifacts) / **localStorage** (로컬)
- **서버 없음** — 완전 클라이언트사이드

### 의존성
React 기본 훅만 사용: `useState`, `useEffect`, `useRef`

런타임 외부 라이브러리 없음. Vite/React는 dev 의존성.

## 디자인 가이드

### 색상 체계
- **Primary**: `#16a34a` (녹색 — 테이블 테두리, 헤더, 숫자태그 기본색)
- **태그 색상** (설정에서 변경 가능):
  - 숫자태그(1~10): `#16a34a` (녹색, 일괄 변경)
  - 개념: `#3B82F6` (파랑)
  - 형태/형태1~3: `#F59E0B` / `#F97316` (앰버/오렌지)
  - 예문: `#22C55E` (초록)
  - 종류: `#EF4444` (빨강)
  - 주의: `#E11D48` (핑크레드)
  - 예시: `#14B8A6` (틸)

### A4 인쇄 규격
- 전체 크기: `740x1046px` (미리보기 = 인쇄 동일)
- 행 수: `30행` 고정
- 행 높이: `27px` 고정
- 컬럼: `1fr 1fr` (50:50), 좌우 구분선 2px 초록
- 인쇄 마진: `@page margin:0` (컨텐츠 자체에 패딩 포함)
- 빈 셀 배경: `#fafafa`, 내용 있는 셀: `#fff`
- 헤더 셀 배경: `#e8f5e9`
