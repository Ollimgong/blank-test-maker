# 백지테스트 메이커

영어학원에서 사용하는 **백지테스트(빈칸 시험지)** 를 편집하고 인쇄할 수 있는 데스크톱 앱.

기존에 한글(HWP) 프로그램으로 제작하던 백지테스트를, 워드처럼 파일을 열고 편집하고 저장하는 방식으로 사용할 수 있도록 Electron + React로 구현.

---

## 프로젝트 구조

```
├── src/
│   ├── blank_test_maker.jsx   # 메인 앱 (React 단일 컴포넌트)
│   └── main.jsx               # Vite 엔트리포인트
├── electron/
│   ├── main.js                # Electron 메인 프로세스
│   └── preload.js             # contextBridge API
├── index.html                 # Vite HTML 엔트리
├── vite.config.js
├── package.json
└── README.md
```

## 작업 폴더 기반 구조

```
작업폴더/
  문법 기초/          ← 하위폴더 = 그룹
    수동태.btm        ← 1파일 = 1단원
    관계대명사.btm
  어휘/
    형용사.btm
```

- `.btm` 파일 하나가 하나의 단원 (시험지)
- 하위 폴더가 그룹 역할
- 다른 선생님과 공유: `.btm` 파일만 주고받으면 됨

## 기능

### 편집
- **에디터 = 프리뷰**: contentEditable 기반 실시간 렌더링. 포커스 시 문법 하이라이팅, 비포커스 시 풀 렌더링
- **2열 테이블**: A4 출력 표와 동일한 너비(684px)로 좌/우 동시 편집
- **A4 미리보기 출력**: A4 전체 미리보기에서 답지만 / 시험지만 / 둘 다 바로 인쇄
- **A/W 전환**: 답지(ANSWER) / 시험지(WORKSHEET) 모드 토글. 일반 텍스트는 시험지에서 숨김
- **자동저장**: 편집 후 1초 뒤 자동으로 파일에 저장
- **BTM 마크업**: 모든 서식을 텍스트 안에 직접 입력 (상세: [`BTM_SPEC.md`](BTM_SPEC.md))
  - `# 헤더` → 섹션 구분 (줄 시작)
  - `## 소제목` → 강조 소제목 (줄 시작)
  - `@태그` → 컬러 뱃지 (자동완성 드롭다운)
  - `[라벨]` → 회색 뱃지
  - `! 텍스트` → 시험지에서도 표시 (줄 시작, 파란색 렌더링)
  - `[[...]]` → 이스케이프
- **왼쪽/오른쪽 독립 이동**: Alt+↑↓로 각 칸 내용을 독립적으로 이동

### 통합 사이드바 & 출력
- **통합 사이드바**: 파일 목록 관리 + 출력 대상 선택이 한 곳에
- **출력 모드 선택**: A(답지) / W(시험지) / A+W(둘 다) 3택
- **선택 출력**: 체크한 파일을 모달에서 썸네일 미리보기 후 PDF 출력
- **사이드바 상시 열림**: 사이드바는 기본 열린 상태, ☰ 버튼으로 수동 토글

### 파일 관리
- **작업 폴더**: 폴더를 선택하면 내부 `.btm` 파일이 사이드바에 표시
- **그룹 = 하위폴더**: 폴더 생성/삭제로 그룹 관리
- **파일 복제/삭제/그룹 이동**: 사이드바 ⋯ 메뉴
- **마지막 폴더 기억**: 앱 재시작 시 자동으로 마지막 작업 폴더 열기

## 데이터 모델

### .btm 파일 (1파일 = 1단원)
```javascript
{
  title: "수동태",
  rows: [{               // 30행 고정
    id,
    l: { text, indent },  // 좌우 동일 구조. 모든 서식은 인라인 마크업으로 text에 포함
    r: { text, indent }
  }]
}
```

### 셀 속성
| 속성 | 설명 |
|------|------|
| `text` | 내용 텍스트 — `#태그`, `[라벨]`, `<소제목>`, `*보임*`, `== 헤더 ==` 인라인 마크업 포함 |
| `indent` | 들여쓰기 레벨 (0/1/2, 현재 미사용) |

## 기술 스택

- **Electron** (데스크톱 앱, 파일 시스템 접근)
- **React** (JSX 단일 파일)
- **Vite** (로컬 개발서버, 빌드)
- **electron-builder** (Windows .exe / macOS .dmg 배포)
- **Pretendard** (웹폰트, CDN)
- **인라인 스타일** (Tailwind 미사용)
- **서버 없음** — 완전 클라이언트사이드

### 의존성
React 기본 훅만 사용: `useState`, `useEffect`, `useRef`

런타임 외부 라이브러리 없음.

## 실행 방법

### 설치
```bash
npm install
```

### 개발
```bash
npm run dev              # Vite HMR + Electron (기본 개발 명령어)
npm run electron:dev     # Vite 빌드 → Electron (프로덕션 빌드 확인용)
```

### 배포 빌드
```bash
npm run dist:win         # Windows .exe 설치파일
npm run dist:mac         # macOS .dmg
```

## A4 인쇄 규격

- 전체 크기: `740x1046px` (미리보기 = 인쇄 동일)
- 행 수: `30행` 고정
- 행 높이: `27px` 고정
- 컬럼: `1fr 1fr` (50:50), 좌우 구분선 2px
- 빈 셀 배경: `#fafafa`, 내용 있는 셀: `#fff`
- 헤더 셀 배경: `#e8efe9`

### 브랜드 컬러
- **Primary Green**: `#00391e` (테이블 테두리, 헤더, ANSWER 모드)
- **Accent Orange**: `#ec6619` (버튼, WORKSHEET 모드, 강조)
