# 백지테스트 메이커

영어학원에서 사용하는 **백지테스트(빈칸 시험지)** 를 편집하고 인쇄할 수 있는 데스크톱 앱.

기존에 한글(HWP) 프로그램으로 제작하던 백지테스트를, 워드처럼 파일을 열고 편집하고 저장하는 방식으로 사용할 수 있도록 Electron + React로 구현.

---

## 프로젝트 구조

```
├── src/
│   ├── blank_test_maker.jsx   # 메인 앱 (React 단일 컴포넌트)
│   └── main.jsx               # Vite 엔트리포인트
├── electron/
│   ├── main.js                # Electron 메인 프로세스
│   └── preload.js             # contextBridge API
├── index.html                 # Vite HTML 엔트리
├── vite.config.js
├── package.json
└── README.md
```

## 작업 폴더 기반 구조

```
작업폴더/
  문법 기초/          ← 하위폴더 = 그룹
    수동태.btm        ← 1파일 = 1단원
    관계대명사.btm
  어휘/
    형용사.btm
```

- `.btm` 파일 하나가 하나의 단원 (시험지)
- 하위 폴더가 그룹 역할
- 다른 선생님과 공유: `.btm` 파일만 주고받으면 됨

## 기능

### 편집
- **편집기**: 2단(왼쪽/오른쪽) 테이블 형식으로 내용 입력
- **실시간 미리보기**: 편집 화면 옆에 A4 비율 미리보기 동시 표시
- **A4 미리보기 출력**: A4 전체 미리보기에서 답지만 / 시험지만 / 둘 다 바로 인쇄
- **ANSWER/WORKSHEET 전환**: 각 셀의 "표시" 버튼으로 마킹하면 WORKSHEET 모드에서 자동 숨김
- **자동저장**: 편집 후 1초 뒤 자동으로 파일에 저장
- **태그 시스템**: 컬러칩 팝업으로 선택 — 숫자태그(1~10)와 일반태그(개념/형태/예문 등)
- **마커(mark)**: 자유 텍스트 접두어 — `[영작]`, `(1)`, `※` 등
- **헤더(H)**: 섹션 헤더 스타일, 왼쪽/오른쪽 독립
- **굵게(B)**: 볼드+밑줄 스타일
- **들여쓰기(┗)**: 레벨 0/1/2, 윗줄의 태그/마커 복제하여 정렬
- **왼쪽/오른쪽 독립 이동**: ▲▼로 각 칸 내용을 독립적으로 이동

### 통합 사이드바 & 출력
- **통합 사이드바**: 파일 목록 관리 + 출력 대상 선택이 한 곳에
- **출력 모드 선택**: A(답지) / W(시험지) / A+W(둘 다) 3택
- **선택 출력**: 체크한 파일을 모달에서 썸네일 미리보기 후 PDF 출력
- **자동 닫힘**: 에디터 셀 포커스 시 사이드바 자동 닫힘, 왼쪽 세로 바 클릭으로 재개

### 파일 관리
- **작업 폴더**: 폴더를 선택하면 내부 `.btm` 파일이 사이드바에 표시
- **그룹 = 하위폴더**: 폴더 생성/삭제로 그룹 관리
- **파일 복제/삭제/그룹 이동**: 사이드바 ⋯ 메뉴
- **마지막 폴더 기억**: 앱 재시작 시 자동으로 마지막 작업 폴더 열기

## 데이터 모델

### .btm 파일 (1파일 = 1단원)
```javascript
{
  title: "수동태",
  rows: [{               // 30행 고정
    id,
    l: { tag, mark, text, vis, bold, hdr, indent },  // 좌우 동일 구조
    r: { tag, mark, text, vis, bold, hdr, indent }
  }],
  settings: { logo, slogan, customFont, customFontName, tags, numTagColor }
}
```

### 셀 속성
| 속성 | 설명 |
|------|------|
| `tag` | 컬러 태그 — 숫자태그(1~10) 또는 일반태그(개념/형태/예문 등) |
| `mark` | 자유 텍스트 마커 — `[영작]`, `(1)` 등 |
| `text` | 내용 텍스트 |
| `vis` | `true`면 WORKSHEET 모드에서도 표시 (false면 숨김) |
| `bold` | `true`면 굵게 + 밑줄 |
| `hdr` | `true`면 헤더 스타일 (초록 배경) |
| `indent` | 들여쓰기 레벨 (0/1/2) |

## 기술 스택

- **Electron** (데스크톱 앱, 파일 시스템 접근)
- **React** (JSX 단일 파일)
- **Vite** (로컬 개발서버, 빌드)
- **electron-builder** (Windows .exe / macOS .dmg 배포)
- **Pretendard** (웹폰트, CDN)
- **인라인 스타일** (Tailwind 미사용)
- **서버 없음** — 완전 클라이언트사이드

### 의존성
React 기본 훅만 사용: `useState`, `useEffect`, `useRef`

런타임 외부 라이브러리 없음.

## 실행 방법

### 설치
```bash
npm install
```

### 개발
```bash
npm run dev              # Vite HMR + Electron (기본 개발 명령어)
npm run electron:dev     # Vite 빌드 → Electron (프로덕션 빌드 확인용)
```

### 배포 빌드
```bash
npm run dist:win         # Windows .exe 설치파일
npm run dist:mac         # macOS .dmg
```

## A4 인쇄 규격

- 전체 크기: `740x1046px` (미리보기 = 인쇄 동일)
- 행 수: `30행` 고정
- 행 높이: `27px` 고정
- 컬럼: `1fr 1fr` (50:50), 좌우 구분선 2px
- 빈 셀 배경: `#fafafa`, 내용 있는 셀: `#fff`
- 헤더 셀 배경: `#e8efe9`

### 브랜드 컬러
- **Primary Green**: `#00391e` (테이블 테두리, 헤더, ANSWER 모드)
- **Accent Orange**: `#ec6619` (버튼, WORKSHEET 모드, 강조)
